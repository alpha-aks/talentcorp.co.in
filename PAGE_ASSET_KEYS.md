## Field Tips

- name: readable title (example: Security Hero)
- key: exact key from this list
- image: upload media
- alt: descriptive alt text
- targetPath: optional page route (example: /security)
- linkUrl: optional external/internal URL if image should be clickable
- order: numeric sort order
